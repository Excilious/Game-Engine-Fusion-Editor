# Game Engine "Fusion" Editor
## A **VERY** basic editor

Some things are unfinished and need to be implemented in the future. The tools only edit the model matrix of said geometry (i.e. Translation, Rotation and Scale matrices). You can edit the geometry vertex however this would be mostly based on the vertex given by the matrices rather than direct editing of the geometry vertex in object space.

Other things will include a texture editor (Albeit being buggy when scaling down or multipling the UV coordinates, perhaps later on it could use a more "better" editing system). Models are supported but would only read .glb/.gltf with textures being read using a format (.imgbuf), making it easier to be read and written rather than using multiple different types of image formats.
